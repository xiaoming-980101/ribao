# 赢日志 (Winner Daily) - 智能日志与周报助手

赢日志是一款专为公司日志考核设计的**本地化、智能化日志生成与管理工具**。

它支持**多模式（正常有任务、无任务维护、技术学习、豆包Prompt生成）**的工作日志生成，提供**7天滚动补填预警**、**全字段适配表单预览（含单独/一键复制）**，以及**本周工作周报智能汇总**功能。内置编辑距离查重算法，保护绩效红线，杜绝任何“抄袭、敷衍、雷同”的抽查风险。

---

## 💾 数据存放在哪里？

您的所有工作日志和配置参数都**100%保存在您的本地计算机上**，安全隐私，绝不上报！
* **本地物理文件数据库路径**：`d:\app\ribao\db.json`
* 只要启动本地后端服务（即双击 `start.bat`），系统写入的每一笔日志都会实时更新至该文件。
* **数据容错与灾备**：即使未开启后端服务，应用也会自动降级将数据存于浏览器的 `LocalStorage` 中，并支持在配置面板中“一键导出 JSON 备份”及“导入还原 JSON”。

---

## 🚀 快速启动指南

在项目根目录下，双击运行 **`start.bat`** 一键启动脚本：
1. **自动依赖检测**：脚本会自动检查并执行 `npm install` 补全所有运行依赖。
2. **多活并行启动**：自动在后台最小化拉起本地 Express 数据库服务器（端口 3001）与 Vite 前端开发服务器（端口 3000）。
3. **自动弹窗打开网页**：等待 3 秒后，自动调用系统默认浏览器打开系统页面：[http://localhost:3000](http://localhost:3000)。

## 🤖 豆包大模型提示词助手

如果您喜欢用大模型（如豆包、Kimi）来生成润色日志，可以使用系统内置的 **“豆包提示词”** 模式：
1. 切换到 `🤖 豆包提示词` 状态。
2. 输入您今日的简单工作（支持不输，不输默认生成日常代码整理及修复细节的提示词）。
3. 点击 **“一键生成豆包 Prompt”**，右侧内容框将显示专门调优、加入正反例对照、防止 AI 官腔的 Prompt。
4. 点击左侧出现的 **“📋 复制 Prompt 并前往豆包”** 按钮，系统会**自动复制提示词到剪贴板，并直接在浏览器中帮您打开豆包网**！
5. 在豆包中粘贴发送后，把生成的高质量“标题”和“内容”贴回右侧表单，点击“保存并记录”即可！
